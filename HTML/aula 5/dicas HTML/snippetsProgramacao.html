Dicas de HTML

Como usar a nav para navegação entre links 

        

    <nav>
      <a href="pagina1.html"> Página 1 </a>
      <a href="pagina2.html"> Página 2 </a>
    </nav>

Como criar um formulário simples 

        

    <form action="http://apiresponse.com" method="post">

      <input type="text" name="nome" placeholder="insira seu nome">
      <input type="number" name="idade" placeholder="insira sua idade">

    
      <!--input do tipo radio tem que ter o mesmo valor no name,
      para serem da mesma família não selecionar os dois ao mesmo tempo-->
    
      <label for="masc">Masculino</label>
      <input type="radio" name="sexo" id="masc"value="masculino">

      <label for="fem">Masculino</label>
      <input type="radio" name="sexo" id="fem" value="feminino">

    </form>
